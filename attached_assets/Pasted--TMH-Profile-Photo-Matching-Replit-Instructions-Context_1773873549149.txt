## Step 3: Matching Logic

For each profile in the TMH founder/operator directory:

1. Take the person's name from the database
2. Normalize it (lowercase, strip prefixes like "Dr.", remove special characters like apostrophes and hyphens)
3. Match it against the photo filenames using the same normalization
4. If a match is found, set that person's profile photo to the corresponding file path (e.g. `/photos/profiles/Sultan_Al_Darmaki.jpg`)
5. If no match is found, leave the existing placeholder/default avatar in place

**Matching rules to handle edge cases:**
- "Dr." prefix in the database should match filenames that start with `Dr_` (e.g. "Dr. Ajlan Al Zaki" matches `Dr_Ajlan_Al_Zaki.jpg`)
- Names with apostrophes like "Sam'an" should match `Saman`
- Names with hyphens like "Al-Jaweini" should match `Al-Jaweini`
- "Mohamed (Nagaty) Aboulnaga" should match `Mohamed_Nagaty_Aboulnaga.jpg` -- strip parentheses
- If the database uses "Dina Sam'an", match to `Dina_Saman.jpg`
- If the database uses "Ghassan R. Haddad", match to `Ghassan_Haddad.jpg` -- drop the middle initial
